GIT TUTORIAL

by PICHARDO

Repositorio

Un repositorio de Git es un almacenamiento virtual de tu proyecto. Te permite guardar versiones del código a las que puedes acceder cuando lo necesites.

git init

Es el comando que utilizamos para inicializar un repositorio.

Working Directory

Se refiere a todos los archivos de los cuales git estará pendiente.

Aquí se encuentran todos los archivos excepto aquellos especificados en el .gitignore

Staging Area

Se refiere a todos los archivos, cuyos cambios estarán incluidos en el próximo commit.

git add . git add filename directory/

Los archivos que se encuentran aquí, tendrán los cambios que se incluirán en el próximo commit

Commit

Se crea una nueva snapshot de nuestro código, con los archivos que se encuentran en la staging area

git commit git commit -m "Nombre del commit"

Aquí se encuentran los cambios ya completados y que queremos separar de las demás versiones de nuestro código.
